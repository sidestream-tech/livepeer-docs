# Tracker Update Rules

`00-master-packet.md` is the live execution tracker.

## Strikethrough Rules

- Strike through an individual task bullet only when that task is complete and the relevant checks were rerun.
- A page is complete only when both `[copy-framework]` and `[authoring-style]` bullets are struck through.
- A section is complete only when every page-level task in that section is struck through.
- When a section is complete, strike through the section title and status line.

## Rules That Do Not Change

- Do not strike through report reference lines.
- Do not convert bullets to checkboxes.
- Do not mark tasks complete based on intent alone.
- Do not close tasks early because a later section might fix them indirectly.

## Completion Notes

When a section closes, add a short completion note that names:

- the checks rerun
- any accepted residual warnings
- any intentionally deferred work that is outside the approved tranche
