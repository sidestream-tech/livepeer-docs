## What you will need

| Requirement | Details |
|---|---|
| **GPU** | NVIDIA with NVENC support (GTX 1060+ for transcoding, RTX 3090+ for AI) |
| **OS** | Linux recommended. macOS/Windows for development only. |
| **Docker** | Docker Engine + NVIDIA Container Toolkit |
| **Wallet** | Ethereum account with ETH and LPT on Arbitrum One |
| **Port** | 8935 TCP open to the public internet |
| **RPC** | Arbitrum One endpoint (Alchemy or Infura free tier) |

For detailed hardware guidance, see [Hardware Reference](/v2/orchestrators/guides/feasibility-and-hardware/hardware-reference).

---

## Quickstart vs Setup guide

| | Quickstart | Setup Guide |
|---|---|---|
| **Goal** | First working node, fast | Production-ready deployment |
| **Install method** | Docker only | Docker, binary, or source |
| **Decisions** | Opinionated defaults | You choose every flag |
| **Time** | ~30 minutes | ~1–2 hours |
| **Best for** | First-time operators, evaluation | Production, custom configurations |

If you need more control — custom binary installs, split orchestrator/transcoder, advanced pricing, or AI pipelines — use the [Setup Guide](/v2/orchestrators/setup/guide) instead.
