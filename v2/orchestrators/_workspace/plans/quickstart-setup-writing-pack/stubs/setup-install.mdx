{/* PAGE STRUCTURE:

## Install go-livepeer

<Tip>Docker is recommended. It matches the quickstart and handles GPU passthrough
with nvidia-container-toolkit.</Tip>

Use <View> components for OS selection (top-right dropdown):

<View title="Docker">
  Steps for Docker install:
  1. Install Docker + nvidia-container-toolkit
  2. docker pull livepeer/go-livepeer:stable
  3. Verify: docker run --rm --gpus all livepeer/go-livepeer:stable -version
  For dual mode: also pull AI runner: docker pull livepeer/ai-runner:latest
</View>

<View title="Linux">
  Steps for Linux binary:
  1. Download release from GitHub (correct archive: livepeer-linux-gpu-amd64.tar.gz)
  2. Verify checksum
  3. Extract and install to /usr/local/bin
  4. Verify: livepeer -version
  Pin to release tag. Community upgrade script link.
</View>

<View title="Windows">
  Steps for Windows:
  1. Download Windows release
  2. Extract to chosen directory
  3. Verify from PowerShell
  Note: AI runner requires Linux. Windows supports video-only via Docker WSL2 or native binary.
</View>

<View title="macOS">
  Steps for macOS:
  1. Download macOS release
  2. Extract and install
  3. Verify
  Note: macOS useful for local dev/CLI only. Production transcoding and AI require Linux.
</View>

## Verify Installation
Verification command per platform. Expected output showing version number.

Link to Configure page.
