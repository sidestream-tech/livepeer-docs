## Common Configuration

### Networking
- -serviceAddr: must be public IP or domain, reachable from internet
- Port 8935: the service port gateways connect to
- Port 7935: CLI and metrics (keep localhost-only unless monitoring externally)
- Firewall: port 8935 must be open inbound

### GPU Selection
- -nvidia flag: specify GPU IDs (use nvidia-smi -L to list)
- Multiple GPUs: comma-separated IDs
- CPU fallback: omit -nvidia for CPU-only transcoding (much slower)

### Pricing
- -pricePerUnit: video pricing in wei per pixel (brief, link to pricing-strategy for full guide)
- AI pricing: set in aiModels.json price_per_unit field
- Link to Config and Optimisation > Pricing Strategy for competitive positioning

### Persistent Data
- Volume mount for /root/.lpData (keystore, data)
- Volume mount for AI models directory (if AI enabled)

## Verify Configuration
Before connecting on-chain, test locally:
```bash
# Check node starts without errors
docker logs <container>
# Check GPU detected
# Check flags applied
```
