## Prerequisites
Checklist with verification commands:
- NVIDIA GPU: `nvidia-smi` (confirm detected, note VRAM)
- Docker: `docker --version`
- nvidia-container-toolkit: `docker run --rm --gpus all nvidia/cuda:12.0-base nvidia-smi`

## Video Transcoding Test

<StyledSteps>

### Step 1: Pull go-livepeer
```bash
docker pull livepeer/go-livepeer:stable
```

### Step 2: Start a local orchestrator (off-chain)
```bash
docker run --gpus all \
  livepeer/go-livepeer:stable \
  -orchestrator -transcoder \
  -network offchain \
  -nvidia 0 \
  -serviceAddr 127.0.0.1:8936
```
Explain: -network offchain means no blockchain, no staking, local only.

### Step 3: Start a local gateway (off-chain, same machine)
In a second terminal:
```bash
docker run \
  livepeer/go-livepeer:stable \
  -gateway \
  -network offchain \
  -orchAddr 127.0.0.1:8936 \
  -rtmpAddr 0.0.0.0:1935 \
  -httpAddr 0.0.0.0:8935
```

### Step 4: Send a test stream
```bash
ffmpeg -re -f lavfi -i testsrc=size=1280x720:rate=30 \
  -c:v libx264 -f flv rtmp://127.0.0.1:1935/test_stream
```
Or use the test video download link if available.

### Step 5: Verify transcoded output
```bash
ffprobe http://127.0.0.1:8935/stream/test_stream.m3u8
```
Check orchestrator logs for: "Got segment" and "Transcode result"

</StyledSteps>

### What Just Happened
Brief diagram: ffmpeg → gateway (RTMP) → orchestrator (GPU transcode) → gateway (HLS out)
Confirm: GPU transcoding works on this machine.

## AI Inference Test

<Note>
Requires 24 GB+ VRAM for diffusion models. If VRAM is 8-16 GB, skip to the
LLM alternative below.
</Note>

<StyledSteps>

### Step 1: Pull AI runner container
```bash
docker pull livepeer/ai-runner:latest
```

### Step 2: Write minimal aiModels.json
One warm model. Use ByteDance/SDXL-Lightning or equivalent fast model.
```json
[
  {
    "pipeline": "text-to-image",
    "model_id": "ByteDance/SDXL-Lightning",
    "warm": true,
    "price_per_unit": 4768371
  }
]
```

### Step 3: Start orchestrator + AI runner (off-chain)
Docker-compose with:
- Orchestrator: -orchestrator -transcoder -aiWorker -aiModels -network offchain -nvidia 0
- AI runner service: livepeer/ai-runner with GPU access

### Step 4: Wait for model download
First start downloads from HuggingFace. Watch logs for "Loaded model" message.
Time: 5-30 min depending on model size and connection.

### Step 5: Test inference locally
```bash
curl -X POST http://localhost:8935/text-to-image \
  -H "Content-Type: application/json" \
  -d '{
    "model_id": "ByteDance/SDXL-Lightning",
    "prompt": "a test image of a mountain",
    "width": 512,
    "height": 512,
    "num_inference_steps": 4
  }'
```
A JSON response with an `images` array confirms AI inference works.

</StyledSteps>

### LLM Alternative (8-16 GB VRAM)
<Accordion title="LLM test for lower-VRAM GPUs" icon="microchip">
Brief: Use Ollama runner instead of livepeer/ai-runner.
Minimal docker-compose with Ollama + Ollama AI Runner.
Test with curl to /llm endpoint.
Cross-ref: llm-pipeline-setup for full details.
</Accordion>

### What Just Happened
Brief: gateway received HTTP request → routed to orchestrator → AI runner loaded
model and processed inference → result returned.
Confirm: AI inference works on this machine.
