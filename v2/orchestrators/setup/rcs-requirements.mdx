## Software

<StyledTable variant="bordered">
  <thead>
    <TableRow header>
      <TableCell header>Requirement</TableCell>
      <TableCell header>Detail</TableCell>
      <TableCell header>Verify</TableCell>
    </TableRow>
  </thead>
  <tbody>
    <TableRow>
      <TableCell>**OS**</TableCell>
      <TableCell>Linux (Ubuntu 22.04+ recommended). <Badge color="blue">Video</Badge> also works on Windows (WSL2) and macOS. <Badge color="purple">AI</Badge> and <Badge color="green">Dual</Badge> require Linux.</TableCell>
      <TableCell>`uname -a`</TableCell>
    </TableRow>
    <TableRow>
      <TableCell>**NVIDIA driver**</TableCell>
      <TableCell>Version 525+ (Linux) or 527+ (Windows)</TableCell>
      <TableCell>`nvidia-smi`</TableCell>
    </TableRow>
    <TableRow>
      <TableCell>**CUDA**</TableCell>
      <TableCell>12.0+ (or use Docker image which bundles CUDA)</TableCell>
      <TableCell>`nvcc --version`</TableCell>
    </TableRow>
    <TableRow>
      <TableCell>**Docker** (recommended)</TableCell>
      <TableCell>Docker Engine + NVIDIA Container Toolkit for GPU passthrough</TableCell>
      <TableCell>`docker run --gpus all nvidia/cuda:12.0-base nvidia-smi`</TableCell>
    </TableRow>
  </tbody>
</StyledTable>

## Network

<StyledTable variant="bordered">
  <thead>
    <TableRow header>
      <TableCell header>Requirement</TableCell>
      <TableCell header>Detail</TableCell>
    </TableRow>
  </thead>
  <tbody>
    <TableRow>
      <TableCell>**Port 8935 TCP**</TableCell>
      <TableCell>Open to the public internet (firewall, router NAT, cloud security group). This is how gateways reach the orchestrator.</TableCell>
    </TableRow>
    <TableRow>
      <TableCell>**Static IP or hostname**</TableCell>
      <TableCell>Required for the service URI that gateways connect to. Dynamic DNS acceptable for home connections.</TableCell>
    </TableRow>
    <TableRow>
      <TableCell>**Bandwidth**</TableCell>
      <TableCell>100 Mbps symmetric minimum. 1 Gbps recommended for multi-stream video transcoding. AI inference requires less bandwidth per job.</TableCell>
    </TableRow>
    <TableRow>
      <TableCell>**Latency**</TableCell>
      <TableCell>Under 50 ms to major regions improves gateway selection probability.</TableCell>
    </TableRow>
  </tbody>
</StyledTable>

## Quick Verification

Run these commands to confirm the prerequisites are met:

```bash icon="terminal" filename="verify-prerequisites"
# GPU visible and driver installed
nvidia-smi

# CUDA version (skip if using Docker - it bundles CUDA)
nvcc --version

# Docker GPU passthrough works
docker run --rm --gpus all nvidia/cuda:12.0-base nvidia-smi

# Port 8935 reachable (run from a DIFFERENT machine)
curl -k https://YOUR_PUBLIC_IP:8935/status
```
