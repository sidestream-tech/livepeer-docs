Example **aiModels.json** (AI pipelines and models; `warm: true` preloads the model into GPU memory):

<CustomCodeBlock
  codeString={`[
  {
    "pipeline": "text-to-image",
    "model_id": "stabilityai/sd-turbo",
    "warm": true
  }
]`}
  language="json"
  icon="brackets-curly"
  filename="aiModels.json"
/>

Create a `models` directory for model data:

<CustomCodeBlock
  codeString="mkdir models"
  language="bash"
  icon="terminal"
/>
