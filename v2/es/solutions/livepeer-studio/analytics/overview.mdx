Livepeer Studio proporciona **audiencia y participación** datos para que puedas medir reproducciones, tiempo de visualización y calidad de reproducción.

## API de audiencia

La [API de audiencia](https://livepeer.studio/docs/api-reference/viewership/get-viewership-metrics) (y endpoints relacionados) proporciona:

- **Visualizaciones en tiempo real** — Número actual de espectadores y tasa de error (requiere Player SDK 4.2.0+).
- **Métricas de uso** — Visitas (intención de reproducción), minutos visualizados. Los datos se actualizan cada 5 minutos aproximadamente; las nuevas subidas pueden tardar hasta una hora en aparecer.
- **Métricas de rendimiento** — Tasa de error, tiempo hasta el primer cuadro (TTFF), relación de rebuffer, salidas antes del inicio.
- **Dimensiones** — Filtrar por video (ID de reproducción, CID), navegador, dispositivo, sistema operativo, geografía, zona horaria, período de tiempo.

## Registro de vistas

- **Livepeer Reproductor** — Las métricas se recopilan automáticamente cuando utiliza el [Livepeer Player](../player).
- **Reproductor personalizado** — Usa la integración `addMediaMetrics`(o equivalente) descrita en las [Livepeer documentación de Studio](https://livepeer.studio/docs) por lo que los eventos de vistas y calidad se informan.

## Otras integraciones

- **Grafana** — Utilice las API de visualización/usabilidad para crear tableros. Consulte las[Livepeer docs de Studio](https://livepeer.studio/docs) para ejemplos.
- **Timeplus** — Uso similar de la API para análisis en tiempo real o histórico.

Para ver las rutas y parámetros exactos de los puntos de conexión, consulte la [referencia de la API de Viewership](https://livepeer.studio/docs/api-reference/viewership/get-viewership-metrics) y páginas relacionadas.
