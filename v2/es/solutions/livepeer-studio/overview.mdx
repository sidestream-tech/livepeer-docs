Livepeer Studio es una puerta de entrada amigable para desarrolladores y una plataforma alojada para la red Livepeer. Úsela para agregar**transmisión en vivo** y **video bajo demanda (VOD)** a su aplicación con un mínimo de configuración.

<Info>
  For Livepeer's hosted real-time StreamDiffusion AI platform **Daydream**, see [Daydream](https://pipelines.livepeer.org/docs) and the [Daydream product page](/solutions/daydream/overview).
</Info>

## Lo que puedes hacer

- **Transmisión en vivo** — Cree transmisiones, ingrese mediante RTMP o WebRTC, reproduzca con WebRTC o HLS de baja latencia, grabe sesiones y transmita simultáneamente a otras plataformas.
- **Vídeo bajo demanda** — Suba activos (TUS o PUT), transcode, reproduzca con el Livepeer Player o cualquier reproductor HLS/MP4, y use activos encriptados opcionalmente.
- **Control de acceso** — Controle la reproducción con webhooks o JWTs (por ejemplo, suscripciones, token-gating).
- **Eventos y análisis** — Utilice webhooks para eventos de transmisión/activo y la API de visualización para métricas de participación y rendimiento.

## SDKs y API

- **SDKs del servidor** — [TypeScript/JavaScript](https://www.npmjs.com/package/livepeer), [Go](https://pkg.go.dev/github.com/livepeer/livepeer-go), [Python](https://pypi.org/project/livepeer/) para llamar a la API de Livepeer Studio.
- **React** — [@livepeer/react](https://www.npmjs.com/package/@livepeer/react): Componentes Player y Broadcast para reproducción y transmisión en el navegador.
- **API** — API REST para transmisiones, activos, reproducción, webhooks, multitransmisión y más. Consulte [Visión general de la API](/solutions/livepeer-studio/reference/api) y las [Livepeer Documentación de la API de Studio](https://livepeer.studio/docs).

## Usar Studio como puerta de enlace

Para usar Livepeer Studio como puerta de enlace a la red Livepeer (por ejemplo, para IA o codificación), consulte [Usar el puerta de enlace Livepeer Studio](/gateways/using-gateways/gateway-providers/livepeer-studio-gateway) en la sección de Puertas de enlace.
