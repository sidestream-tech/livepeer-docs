## Botones y Acciones

### DownloadButton

Botón de descarga interactivo con carga diferida para optimización del rendimiento.

**Importar:**
```jsx
import { DownloadButton } from "/snippets/components/primitives/buttons.jsx";
```

**Props:**

| Prop | Tipo | Predeterminado | Requerido | Descripción |
|------|------|---------|----------|-------------|
| `label` | `string` | `"Download"` | No | Etiqueta de texto del botón |
| `icon` | `string` | `"download"` | No | Nombre del icono izquierdo |
| `downloadLink` | `string` | - | **Sí** | URL del archivo para descargar |
| `rightIcon` | `string` | `""` | No | Nombre del icono derecho opcional |
| `border` | `boolean` | `false` | No | Si se muestra un borde alrededor del botón |

**Ejemplos:**

<Tabs>
  <Tab title="Basic">
    <DownloadButton downloadLink="https://example.com/file.pdf" />
  </Tab>
  <Tab title="Custom Label">
    <DownloadButton label="Download SDK" downloadLink="https://example.com/sdk.zip" />
  </Tab>
  <Tab title="With Icons">
    <DownloadButton 
      label="Download PDF" 
      downloadLink="https://example.com/file.pdf" 
      icon="file-pdf"
      rightIcon="arrow-right"
      border={true}
    />
  </Tab>
  <Tab title="Code">
```jsx
<DownloadButton downloadLink="https://example.com/file.pdf" />

<DownloadButton 
  label="Download PDF" 
  downloadLink="https://example.com/file.pdf" 
  icon="file-pdf"
  rightIcon="arrow-right"
  border={true}
/>
```
  </Tab>
</Tabs>

### CustomCallout

Una caja de llamada con estilo, con icono y colores personalizables. Convierte automáticamente los colores en hex a rgba para un manejo adecuado de la opacidad.

**Importar:**
```jsx
import { CustomCallout } from "/snippets/components/primitives/links.jsx";
```

**Propiedades:**

| Prop | Tipo | Predeterminado | Requerido | Descripción |
|------|------|---------|----------|-------------|
| `children` | `ReactNode` | - | **Sí** | Contenido para mostrar en la llamada |
| `icon` | `string` | `"lightbulb"` | No | Nombre del icono a mostrar |
| `color` | `string` | acento de tema | No | Color principal para el icono, borde y fondo |
| `iconSize` | `number` | `16` | No | Tamaño del icono en píxeles |
| `textSize` | `string` | `"0.875rem"` | No | Tamaño de fuente para el contenido de texto |
| `textColor` | `string` | coincide con `color` | No | Color del texto (predeterminado para coincidir con el color del icono) |

**Ejemplos:**

<Tabs>
  <Tab title="Info">
    <CustomCallout icon="info" color="#3b82f6">
      This is an informational message with a blue info icon.
    </CustomCallout>
  </Tab>
  <Tab title="Warning">
    <CustomCallout icon="warning" color="#ef4444">
      Warning: This action cannot be undone.
    </CustomCallout>
  </Tab>
  <Tab title="Success">
    <CustomCallout icon="check-circle" color="#22c55e">
      Success! Your changes have been saved.
    </CustomCallout>
  </Tab>
  <Tab title="Code">
```jsx
<CustomCallout icon="info" color="#3b82f6">
  Important information message
</CustomCallout>

<CustomCallout icon="warning" color="#ef4444">
```

  </Tab>

## Primitivas de tabla

Componentes de tabla para mostrar datos estructurados sin estilos en línea.

### StyledTable

Componente de tabla con conciencia de tema y soporte para variantes.

**Importar:**
```jsx
import { StyledTable, TableRow, TableCell } from "/snippets/components/layout/tables.jsx";
```

**Props:**

| Prop | Tipo | Predeterminado | Requerido | Descripción |
|------|------|---------|----------|-------------|
| `children` | `ReactNode` | - | **Sí** | Contenido de la tabla (thead, tbody, etc.) |
| `variant` | `string` | `"default"` | No | Variante de tabla: "default" | "bordered" | "minimal" |
| `style` | `object` | `{}` | No | Estilos en línea adicionales |

**Ejemplos:**

<Tabs>
  <Tab title="Default Table">
    <StyledTable>
      <thead>
        <TableRow header>
          <TableCell header>Name</TableCell>
          <TableCell header align="center">Status</TableCell>
        </TableRow>
      </thead>
      <tbody>
        <TableRow>
          <TableCell>Item 1</TableCell>
          <TableCell align="center">Active</TableCell>
        </TableRow>
        <TableRow>
          <TableCell>Item 2</TableCell>
          <TableCell align="center">Pending</TableCell>
        </TableRow>
      </tbody>
    </StyledTable>
  </Tab>
  <Tab title="Code">
```jsx
<StyledTable variant="bordered">
  <thead>
    <TableRow header>
      <TableCell header>Header 1</TableCell>
      <TableCell header align="center">Header 2</TableCell>
    </TableRow>
  </thead>
  <tbody>
    <TableRow>
      <TableCell>Data 1</TableCell>
      <TableCell align="center">Data 2</TableCell>
    </TableRow>
  </tbody>
</StyledTable>
```
  </Tab>
</Tabs>

**Props de TableRow:**

| Prop | Tipo | Predeterminado | Requerido | Descripción |
|------|------|---------|----------|-------------|
| `children` | `ReactNode` | - | **Sí** | Celdas de tabla |
| `header` | `boolean` | `false` | No | ¿Es una fila de encabezado? |
| `hover` | `boolean` | `false` | No | Habilitar efecto de paso del mouse |
| `style` | `object` | `{}` | No | Estilos en línea adicionales |

**Props de TableCell:**

| Prop | Tipo | Predeterminado | Requerido | Descripción |
|------|------|---------|----------|-------------|
| `children` | `ReactNode` | - | **Sí** | Contenido de celda |
| `align` | `string` | `"left"` | No | Alineación del texto: "izquierda" | "centro" | "derecha" |
| `header` | `boolean` | `false` | No | ¿Es una celda de encabezado? |
| `style` | `object` | `{}` | No | Estilos en línea adicionales |
