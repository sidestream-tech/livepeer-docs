## Integración de CoinGecko

### CoinGeckoExchanges

Obtiene y muestra dinámicamente las intercambios que admiten una moneda desde la API de CoinGecko. Muestra una tabla ordenable de intercambios (CEX y DEX) con pares de negociación, puntuaciones de confianza y enlaces.

**Importar:**
```jsx
import { CoinGeckoExchanges } from "/snippets/components/integrations/coingecko.jsx";
```

**Props:**

| Prop | Tipo | Predeterminado | Requerido | Descripción |
|------|------|---------|----------|-------------|
| `coinId` | `string` | `"arbitrum"` | No | El ID de moneda de CoinGecko (por ejemplo, "arbitrum", "ethereum", "bitcoin") |

**Características:**
- Recupera automáticamente los datos de intercambio de la API de CoinGecko
- Muestra los intercambios en una tabla ordenable
- Muestra el tipo de intercambio (CEX/DEX), pares de comercio y puntuaciones de confianza
- Incluye enlaces a las páginas de comercio
- Soporta la ordenación por nombre o tipo
- Estados de carga y error

**Ejemplos:**

<Tabs>
  <Tab title="Default (Arbitrum)">
    <CoinGeckoExchanges />
  </Tab>
  <Tab title="Ethereum">
    <CoinGeckoExchanges coinId="ethereum" />
  </Tab>
  <Tab title="Code">
```jsx
<CoinGeckoExchanges coinId="arbitrum" />
<CoinGeckoExchanges coinId="ethereum" />
<CoinGeckoExchanges coinId="bitcoin" />
```
  </Tab>
</Tabs>

**Nota:** Este componente realiza una llamada de API en vivo a CoinGecko. Asegúrese de que no esté excediendo los límites de frecuencia si utiliza múltiples instancias en una página.
