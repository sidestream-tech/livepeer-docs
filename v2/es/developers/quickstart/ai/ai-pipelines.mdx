# Trabajos de IA y flujos de trabajo de inicio rápido

Comience aquí si desea enviar trabajos de inferencia de IA a través de una puerta de enlace Livepeer.

## Elija un inicio rápido

- [Inicio rápido para trabajos de IA](./ai-jobs) - Flujo de solicitud de extremo a extremo usando la API de la puerta de enlace de IA (superficie de punto de conexión validada por GitHub; aún se requiere aprobación del flujo de interesados).
- [Visión general de las pipelines de IA](/developers/ai-pipelines/overview) - Visión general del ecosistema de la cadena de procesamiento de IA en Livepeer.
- [BYOC (Trae tu propio cálculo)](/developers/ai-pipelines/byoc) - Ejecuta tu propio cálculo y conéctalo a la red de IA Livepeer.

## Lo que cubre esta sección

- Enviar solicitudes a la API del gateway de IA
- Elegir un punto de conexión de tarea de IA compatible (por ejemplo, `text-to-image`)
- Autenticación y URLs base
- Solución de problemas y pasos siguientes (BYOC, en tiempo real, tuberías avanzadas)

## Notas

- Este centro de inicio rápido es intencionalmente corto. El flujo de trabajo ejecutable está en [inicio rápido de Jobs de IA](./ai-jobs).
- La precedencia de las fuentes para el inicio rápido de trabajos de IA es GitHub primero, luego la corroboración de DeepWiki, luego los documentos/especificaciones locales del repositorio.
