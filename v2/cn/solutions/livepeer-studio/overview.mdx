Livepeer Studio 是一个开发者友好的网关和托管平台，用于 Livepeer 网络。使用它可以将**实时流媒体** 和 **点播 (VOD)** 添加到您的应用程序，设置简单。

<Info>
  For Livepeer's hosted real-time StreamDiffusion AI platform **Daydream**, see [Daydream](https://pipelines.livepeer.org/docs) and the [Daydream product page](/solutions/daydream/overview).
</Info>

## 您可以执行的操作

- **直播流** - 创建流，通过 RTMP 或 WebRTC 进行采集，使用低延迟的 WebRTC 或 HLS 进行播放，记录会话，并多路传输到其他平台。
- **按需视频** - 上传资源（TUS 或 PUT），转码，使用 Livepeer 播放器或任何 HLS/MP4 播放器进行播放，并可选择使用加密资源。
- **访问控制** - 通过 Webhook 或 JWT（例如订阅、代币门控）限制播放。
- **事件和分析** - 使用 Webhook 来处理流/资产事件，以及观看人数 API 来获取参与度和性能指标。

## SDK 和 API

- **服务器 SDK** - [TypeScript/JavaScript](https://www.npmjs.com/package/livepeer), [Go](https://pkg.go.dev/github.com/livepeer/livepeer-go), [Python](https://pypi.org/project/livepeer/)用于调用 Livepeer Studio API。
- **React** - [@livepeer/react](https://www.npmjs.com/package/@livepeer/react)：用于播放和浏览器内流媒体的播放器和广播组件。
- **API** - 流、资产、播放、webhooks、多流等的 REST API。参见 [API 概述](/solutions/livepeer-studio/reference/api) 和 [Livepeer Studio API 文档](https://livepeer.studio/docs).

## 使用 Studio 作为网关

要将 Livepeer Studio 用作 Livepeer 网络的网关（例如用于 AI 或转码），请参阅 [使用 Livepeer Studio 网关](/gateways/using-gateways/gateway-providers/livepeer-studio-gateway) 在网关部分。
