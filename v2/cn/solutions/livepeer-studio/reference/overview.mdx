# API 参考

欢迎来到 Livepeer Studio API 参考！在这里，您将找到 Livepeer Studio API 上所有公开的端点，学习如何使用它们以及它们返回的内容。

Livepeer Studio API 采用 REST 架构，具有可预测的面向资源的 URL，接受 JSON 请求体，返回 JSON 编码的响应，并使用标准的 HTTP 响应代码、认证和动词。

## 基础 URL

所有 API 请求都应发送到：

```
https://livepeer.studio/api
```

## 认证

所有请求都需要使用 API 密钥进行身份验证。请参阅 [身份验证指南](../get-started/authentication) 以获取详细信息。

## 响应格式

所有响应均为 JSON 编码。成功响应返回一个 `200 OK` 响应体中包含请求的数据的状态码。

 错误响应包含一个错误对象，其中包含有关哪里出错的详细信息。
