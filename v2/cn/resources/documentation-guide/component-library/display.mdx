## 视频组件

### YouTubeVideo

嵌入带有可选标题和作者署名的 YouTube 视频。在 Frame 组件内的响应式 iframe 中显示 YouTube 视频。

**导入：**
```jsx
import { YouTubeVideo } from "/snippets/components/content/video.jsx";
```

**属性：**

| 属性 | 类型 | 默认值 | 必填 | 描述 |
|------|------|---------|----------|-------------|
| `embedUrl` | `string` | - | **是** | YouTube 嵌入 URL（例如，"https://www.youtube.com/embed/VIDEO_ID" |
| `title` | `string` | `""` | 否 | 视频标题，用于辅助功能和字幕显示 |
| `author` | `string` | `""` | 否 | 要在字幕中显示的作者名称 |
| `hint` | `string` | `""` | 否 | 可选的提示文本以显示 |
| `caption` | `string` | - | 否 | 自定义标题（覆盖作者/标题格式） |

**示例：**

{/* YouTubeVideo commented out - component bug: convertToPrivacyEnhancedUrl is not defined */}

### 视频

带有 Frame 包装器的自托管视频组件。在 Frame 组件内显示一个自托管的视频文件。

**导入：**
```jsx
import { Video } from "/snippets/components/content/video.jsx";
```

**属性：**

| 属性 | 类型 | 默认值 | 必需 | 描述 |
|------|------|---------|----------|-------------|
| `src` | `string` | - | **是** | 视频文件路径（例如："https://raw.githubusercontent.com/livepeer/docs/docs-v2-assets/snippets/assets/media/videos/HeroBackground.mp4""） |
| `title` | `string` | `""` | 否 | 用于辅助功能的视频标题 |
| `author` | `string` | `""` | 否 | 要与麦克风图标一起显示的作者名称 |
| `caption` | `string` | - | 否 | 可选的标题文字（如果作者提供了，将使用图标样式） |
| `href` | `string` | `""` | 否 | 作者的可选链接URL |
| `controls` | `boolean` | `true` | 否 | 是否显示视频控件 |
| `autoPlay` | `boolean` | `false` | 否 | 是否自动播放视频 |
| `loop` | `boolean` | `false` | 否 | 是否循环播放视频 |
| `muted` | `boolean` | `false` | 否 | 是否静音视频 |
| `children` | `ReactNode` | - | 否 | 要渲染在 Frame 内部的可选子元素 |

**示例：**

<Tabs>
  <Tab title="Basic">
    <Video src="https://raw.githubusercontent.com/livepeer/docs/docs-v2-assets/snippets/assets/media/videos/HeroBackground.mp4" title="Demo Video" />
  </Tab>
  <Tab title="Autoplay Loop">
    <Video 
      src="https://raw.githubusercontent.com/livepeer/docs/docs-v2-assets/snippets/assets/media/videos/HeroBackground.mp4" 
      title="Demo"
      autoPlay
      muted
      loop
    />
  </Tab>
  <Tab title="With Author">
    <Video 
      src="https://raw.githubusercontent.com/livepeer/docs/docs-v2-assets/snippets/assets/media/videos/livepeer-founders-post.mp4"
      author="Doug Petkanics"
      title="Livepeer Vision"
    />
  </Tab>
  <Tab title="Code">
```jsx
<Video src="https://raw.githubusercontent.com/livepeer/docs/docs-v2-assets/snippets/assets/media/videos/HeroBackground.mp4" title="Demo Video" />

<Video 
  src="https://raw.githubusercontent.com/livepeer/docs/docs-v2-assets/snippets/assets/media/videos/HeroBackground.mp4"
  autoPlay
  muted
  loop
  author="Author Name"
  title="Video Title"
/>
```
  </Tab>
</Tabs>

| 属性 | 类型 | 默认 | 必需 | 描述 |

| `title` | `string` | - | 否 | 叠加在视频上的标题文本 |
| `subtitle` | `string` | - | 否 | 叠加在视频上的副标题文本 |
| `arrow` | `boolean` | `false` | 否 | 是否显示箭头指示器 |

| `style` | `object` | `{}` | 否 | 容器的附加样式 |

<Tabs>

```jsx
<TitledVideo
  src="https://raw.githubusercontent.com/livepeer/docs/docs-v2-assets/snippets/assets/media/videos/HeroBackground.mp4"

```
  </Tab>

### ShowcaseCards

带有搜索、过滤和分页的交互式展示卡片组件。

**导入：**
```jsx
import { ShowcaseCards } from "/snippets/components/data/showcaseCards.jsx";
```

**属性：**

| 属性 | 类型 | 默认 | 必需 | 描述 |
|------|------|---------|----------|-------------|
| `items` | `Array` | `[]` | 否 | 展示项对象数组 |
| `limit` | `number` | `null` | 否 | 可选的项目数量限制 |
| `pageSize` | `number` | `10` | 无 | 每页项目数量 |

**项目对象结构:**
- `title` - 卡片标题
- `subtitle` - 卡片副标题
- `description` - 卡片描述
- `href` - 链接 URL
- `categoryTags` - 类别字符串数组
- `productTags` - 产品字符串数组
- `mediaSrc` - 图像/视频源
- `mediaType` - "image" 或 "video"
- `logo` - Logo 图像 URL
- `links` - 链接对象数组

**示例:**

<Tabs>
  <Tab title="Code">
```jsx
const items = [
  {
    title: "Project Name",
    subtitle: "Subtitle",
    description: "Description",
    href: "/project",
    categoryTags: ["Apps"],
    productTags: ["Livepeer"],
    mediaSrc: "/images/project.png",
    mediaType: "image"
  }
];

<ShowcaseCards items={items} limit={10} pageSize={6} />
```
  </Tab>
</Tabs>
