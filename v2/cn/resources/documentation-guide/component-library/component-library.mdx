## 如何使用组件

在 MDX 文件中使用绝对路径导入组件，路径从`/snippets/`:

```jsx
import { YouTubeVideo } from "/snippets/components/content/video.jsx";
import { GotoCard, GotoLink } from "/snippets/components/primitives/links.jsx";
import { CustomCodeBlock } from "/snippets/components/content/code.jsx";
```

## 基础组件

在整个文档中使用的基本 UI 构建模块。

### 自定义分隔线

带有Livepeer品牌图标在两端的装饰性水平分隔线。可选择显示居中文字。

<Tabs>
  <Tab title="Live Example">
    <CustomDivider />
    <CustomDivider middleText="Section Break" />
  </Tab>
  <Tab title="Code">
```jsx
import { CustomDivider } from "/snippets/components/primitives/divider.jsx";

<CustomDivider />
<CustomDivider middleText="Section Break" />
<CustomDivider middleText="OR" color="var(--accent)" />
```
  </Tab>
  <Tab title="Props">
    | Prop | Type | Default | Description |
    |------|------|---------|-------------|
    | `middleText` | `string` | `""` | Text to display in the center |
    | `color` | `string` | `"var(--border)"` | Color for the middle text |
    | `style` | `object` | `{}` | Additional CSS styles |
  </Tab>
</Tabs>

### Frame 模式组件（PageHeader, H1-H6, P, Divider）

      description="This is a description of the page content"

    <H1 icon="rocket">Main Heading</H1>
