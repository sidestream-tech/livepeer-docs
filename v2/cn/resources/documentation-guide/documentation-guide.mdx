## 网站布局与导航

### 标题栏功能

标题栏提供对关键工具的快速访问：

- **搜索栏 / AI 助手** - 位于每页的顶部。输入您的问题或搜索词以找到相关内容或获取 AI 驱动的答案。
- **版本选择器** - 在 v2（当前）和 v1（旧版）文档之间切换。所有历史信息都保留在 v1 中。
- **社交媒体图标** - 快速链接至 Livepeer 的社交媒体和社区频道。

### 顶部导航标签

主要导航根据用户角色和兴趣分为标签：

- **首页** - 概述、介绍和入门
- **关于** - 核心概念、协议细节和思维模型
- **社区** - 社区资源、活动和参与
- **开发者** - 在 Livepeer 上构建，API、SDK 和指南
- **网关** - 运行网关基础设施
- **协调者** - 设置和运行协调者节点
- **委托人** - 质押和委托指南
- **资源** - 参考中心、术语表和附加资源

每个标签页包含:
- **门户页面** - 带有快速介绍和关键链接的首页
- **分组** - 组织好的部分（概述、入门、构建、工具、指南）
- **页面** - 单个文档页面

### 左侧边栏导航

侧边栏提供特定于部分的导航：

- **锚点** - 快速访问快速入门和参考中心
- **主标签导航** - 当前标签的可展开组和页面
- **语言选择器** - 当前仅支持英语（计划支持多语言）
- **主题选择器** - 在浅色和深色主题之间切换

<Tip>
You can collapse the sidebar by clicking the menu icon to maximise reading space.
</Tip>

### 页面布局

页面使用各种布局组件来组织信息:

- **标签页** - 按上下文（例如，AI 与视频，链上与链下）分离内容
- **视图** - 根据操作系统或用户路径显示不同的内容
- **步骤** - 用于流程的顺序说明
- **卡片组** - 用于门户、中心和相关内容的视觉分组
- **折叠面板** - 可展开的区域，用于详细信息
- **要点提示** - 重要注意事项、技巧、警告和信息框

## 查找信息

### 使用搜索

页眉中的搜索栏提供：

- **语义搜索** - 理解上下文和含义，而不仅仅是关键词
- **即时结果** - 键入时查看建议
- **页面预览** - 匹配内容的快速预览
- **关键词匹配** - 具有相关元数据关键词的页面会优先显示

### 使用 AI 助手

AI 助手（与搜索栏集成）可以：

- 回答关于 Livepeer 的问题
- 解释概念和术语
- 帮助您找到相关文档
- 提供代码示例和指导

<Tip>
The AI Assistant uses the documentation content to provide accurate, up-to-date answers. For the latest information, always refer to the official documentation pages.
</Tip>

### 导航提示

1. **从门户开始** - 每个标签页都有一个门户页面，提供概述和关键入口点
2. **使用导航路径** - 导航路径显示您当前的位置
3. **查看相关页面** - 很多页面包含相关内容的链接
4. **遵循用户流程** - 内容按逻辑性的从零到英雄的流程组织

## 文档功能

### 版本切换

原始 v1 文档的所有文档都已保留。使用标题中的版本选择器来：

- 访问旧版内容
- 比较 v1 和 v2 的方法
- 查找历史信息

### 主题选择

选择您偏好的观看体验:

- **深色主题**（默认）- 优化用于低光环境观看
- **浅色主题**- 传统的浅色背景

<Tip>
Light theme may have some styling inconsistencies as dark mode is the primary design focus.
</Tip>

### 响应式设计

该文档完全响应式，可在以下设备上使用：

- 台式计算机
- 平板电脑
- 移动设备

## 获取帮助

如果您找不到所需内容：

1. **使用搜索** - 尝试不同的搜索词或问题
2. **询问 AI 助手** - 获得即时答案
3. **查看相关页面** - 查看页面底部的链接
4. **按标签浏览** - 通过相关标签导航以满足您的使用场景
5. **提供反馈** - 在任何页面使用反馈机制
