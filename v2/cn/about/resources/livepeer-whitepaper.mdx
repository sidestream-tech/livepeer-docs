Livepeer的原始白皮书（发布于2017年）概述了一个完全去中心化的实时视频流网络的宏伟设计。
目标是构建 _“世界开放的视频基础设施”_，任何人都可以按需贡献或访问视频编码和传输。

Livepeer是区块链领域最早的DePIN（去中心化公共基础设施网络）项目之一，随着AI重塑行业以及我们的工作、协作和沟通方式，这一区块链用例今天变得更加重要。

白皮书中概述的关键理念包括：
- **激励参与** 使用原生代币（Livepeer Token，LPT）和用于视频转码工作的无信任验证机制。
在白皮书的设计中，任何运行节点的人都可以执行视频转码（重新格式化视频以实现高效流式传输），并以LPT获得奖励，同时验证者和博弈论协议将确保输出正确，无需中心化权威。

- **无信任验证：** 一个突出的提议是使用 Truebit（去中心化计算预言机）作为“黑箱”来验证转码后的视频片段。转码节点会在链上提交结果，基于 Truebit 的挑战可以检测错误输出，从而在无需人工监督的情况下确保正确性。

- **开放参与：** 白皮书设想了一个完全开放的转码器（提供计算的节点）和广播者（提交视频的用户）市场，通过智能合约进行交互。不会有内置的中心服务器——“无需依赖任何中心方或公司来维持网络的持续运行”。

- **通过代币实现的经济安全：** LPT 被引入作为一种协调机制——节点会质押 LPT 作为履约保证金，以获得转码视频的权利，使其激励与诚实工作保持一致。该概念至今仍然存在：质押的代币充当针对恶意行为的抵押品，类似于保证金。

<Tip>Did you know? Livepeer’s token distribution had no [ICO](https://messari.io/report/merkle-mine) LPT was distributed in a novel way called a [Merkle Mine](https://github.com/livepeer/merkle-mine), allowing a wide set of participants to claim tokens at network launch. <br/> You can view the [github code here](https://github.com/livepeer/merkle-mine).</Tip>

### Livepeer 现在
Livepeer 如今既继承又扩展了原始白皮书的愿景。由加密代币驱动的去中心化视频网络的核心理念已在生产中得到验证——Livepeer 处理数百万分钟的视频，并在五大洲拥有数百名节点运营商。关键技术转变，如链下扩展（Streamflow，Arbitrum）以及向基于 GPU 的 AI 工作负载的扩展，使项目始终站在“开放视频基础设施”意义的前沿。

最初作为去中心化直播流协议的项目已发展为实时媒体计算的通用平台。
