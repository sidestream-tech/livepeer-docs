#### 决策树
加入GPU池 vs. 运行自己的Orchestrator

{/* Add your GPU to a pool if:

You want low‑touch earnings

You want to avoid crypto governance and staking

You are testing Livepeer for the first time

You operate a small number of GPUs

Run an orchestrator if:

You want to build a long‑term business

You are comfortable with DevOps, and on-chain actions like staking

You want maximum upside

You want to build a reputation as an Orchestrator / GPU provider or run your own pool or be a delegator

You plan to aggregate significant GPU capacity */}


如果你想要轻松、低维护的收益（这只是一个爱好），请将你的GPU添加到池中

- 你不想处理正常运行时间、路由、监控和升级
- 你不想处理正常运行时间、路由、监控和升级
- 您不希望担心质押、委托或定价配置
- 您第一次测试 Livepeer
- 您不想处理质押 LPT 或担心罚没
- 您有 1–5 块 GPU

如果要运行一个协调器则：

- 您希望完全控制您的 GPU 的操作
- 你想建立一个企业
- 你对运营、链上操作和质押感到熟悉（或者愿意花时间学习）
- 你想要最大的回报，并且愿意质押 LPT
- 你计划聚合许多GPU
- 你希望对治理、定价和委托有完全的发言权
- 你有长期的计划
