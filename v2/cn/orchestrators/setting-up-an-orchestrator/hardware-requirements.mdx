## 网络和运维

- **延迟：** &lt;50毫秒到主要地区有助于流媒体和网关选择。
- **生产环境：**静态IP，反向代理（例如nginx），TLS，防火墙规则。
- **监控：**Prometheus，Grafana，NVIDIA DCGM导出器；跟踪GPU利用率，VRAM，片段/作业成功率。

## 上线前清单

- [ ] GPU可通过`nvidia-smi`
- [ ] Docker可见GPU（"`--gpus all`)
- [ ] CUDA 功能
- [ ] 端口开放（例如 8935）
- [ ] 稳定 Arbitrum RPC
- [ ] 监控已配置
