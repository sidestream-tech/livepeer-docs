## GWID（网关向导）

<Warning>
  {' '}
  This is a community contributed project. Please contact the team for help & support.{' '}
</Warning>

GWID 是一个完全托管的 DevOps 平台，适用于 Livepeer。它提供了一种简单易用的方法来部署和管理 Livepeer 网关，并且还提供了与其他服务的轻松集成，以测试网关。
