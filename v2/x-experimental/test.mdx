Use this page to test the **Mintlify AI assistant** (the docs chat).

## How to test the assistant

1. **Open the assistant** — Use the chat button in the docs UI (bottom-right or in the nav).
2. **Try default questions** — If configured in the Mintlify dashboard, suggested prompts appear; click one or type a doc question.
3. **Check behavior** — The assistant answers from the docs (RAG). You can test citations, deflection, and custom default questions here.

Configuration (default questions, knowledge base, search sites) is done in the [Mintlify dashboard](https://dashboard.mintlify.com) under Assistant settings, not in this repo.
