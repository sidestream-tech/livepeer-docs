## Comment utiliser les composants

Importez les composants dans vos fichiers MDX en utilisant des chemins absolus à partir de `/snippets/`:

```jsx
import { YouTubeVideo } from "/snippets/components/content/video.jsx";
import { GotoCard, GotoLink } from "/snippets/components/primitives/links.jsx";
import { CustomCodeBlock } from "/snippets/components/content/code.jsx";
```

## Primitives

Blocs de base de l'interface utilisateur utilisés tout au long de la documentation.

### CustomDivider

Un séparateur horizontal décoratif avec des icônes de marque Livepeer des deux côtés. Affiche éventuellement du texte centré.

<Tabs>
  <Tab title="Live Example">
    <CustomDivider />
    <CustomDivider middleText="Section Break" />
  </Tab>
  <Tab title="Code">
```jsx
import { CustomDivider } from "/snippets/components/primitives/divider.jsx";

<CustomDivider />
<CustomDivider middleText="Section Break" />
<CustomDivider middleText="OR" color="var(--accent)" />
```
  </Tab>
  <Tab title="Props">
    | Prop | Type | Default | Description |
    |------|------|---------|-------------|
    | `middleText` | `string` | `""` | Text to display in the center |
    | `color` | `string` | `"var(--border)"` | Color for the middle text |
    | `style` | `object` | `{}` | Additional CSS styles |
  </Tab>
</Tabs>

### Composants en mode Frame (PageHeader, H1-H6, P, Divider)

Composants d'en-tête et de texte personnalisés pour le mode Mintlify. Ces composants fournissent des en-têtes stylisés qui fonctionnent correctement en mode cadre, où les en-têtes markdown par défaut peuvent ne pas s'afficher correctement.

<Tabs>
  <Tab title="Live Example">
    <PageHeader 
      title="Page Title" 
      subtitle="Page Subtitle"
      description="This is a description of the page content"
    />
    <H1 icon="rocket">Main Heading</H1>
    <H2 icon="star">Section Heading</H2>
    <H3>Subsection</H3>
    <P icon="info-circle">This is a paragraph with an icon.</P>
    <Divider />
  </Tab>
