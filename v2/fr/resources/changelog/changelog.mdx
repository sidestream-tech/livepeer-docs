# Journal des modifications de la documentation

Ce journal des modifications suit les mises à jour importantes de la documentation Livepeer. Pour les changements de protocole, voir le [Livepeer Propositions d'amélioration (LIPs)](https://github.com/livepeer/LIPs) et [Livepeer Forum](https://forum.livepeer.org).

<Note>
  **Planned:** This changelog will be automatically populated from GitHub releases and commits via an n8n pipeline. For now, it's manually maintained.
</Note>

---

> Nouvelles versions et améliorations

<Update label="February 2026" tags={["Governance", "Contributions"]} rss={{ title: "Documentation Governance and Contribution Workflow", description: "Added comprehensive governance documentation, contribution guide, and CODEOWNERS" }}>
  ## Documentation Governance and Contribution Workflow

- Added comprehensive [contribution guide](../documentation-guide/contribute-to-the-docs) with step-by-step PR workflow
- Created [governance documentation](../../internal/governance) covering review process, ownership, and ticketing
- Updated CODEOWNERS with section-specific ownership
- Added PR template for documentation contributions
- Established review SLAs and process

These changes make it easier for the community to contribute and ensure consistent quality across all documentation.

</Update>

<Update label="February 2026" tags={["Content", "Structure"]} rss={{ title: "Documentation v2 Structure Complete", description: "Completed information architecture and navigation structure for v2 documentation" }}>
  ## Documentation v2 Structure Complete

- Implemented complete information architecture with tabs, anchors, and groups
- Organized content by stakeholder personas (Developers, Gateways, Orchestrators, Delegators)
- Separated AI Jobs and Transcoding Jobs documentation
- Added style guides for About, Developers, and Orchestrators sections
- Created component library documentation

The new structure provides clearer navigation and better organization for all user types.

</Update>

<Update label="January 2026" tags={["AI", "Features"]} rss={{ title: "AI Assistant Integration", description: "Integrated Mintlify AI assistant for natural language search and documentation help" }}>
  ## AI Assistant Integration

- Integrated Mintlify AI assistant for embedded natural-language search
- Added semantic headings and structured metadata for LLM parsing
- Improved documentation discoverability with AI-powered search
- Enhanced user experience with contextual help

Users can now ask questions about Livepeer documentation directly in the docs interface.

</Update>

<Update label="2025" tags={["Migration", "Architecture"]} rss={{ title: "Documentation v2 Migration", description: "Migrated documentation to v2 structure with improved IA and content organization" }}>
  ## Documentation v2 Migration

- Migrated from v1 to v2 documentation structure
- Redesigned information architecture with stakeholder-focused navigation
- Consolidated and reorganized content by topic and persona
- Added redirects from v1 to v2 URLs
- Improved content discoverability and organization

This migration provides a better foundation for documentation growth and community contributions.

</Update>

## Comment lire ce journal des modifications

- **Étiquettes** indiquent le type de modification : `Content`, `Structure`, `Features`, `Governance`, `Contributions`, `AI`, `Migration`
- **RSS** entrées fournissent des métadonnées lisibles par machine pour les flux du journal des modifications
- **Mises à jour** sont organisées chronologiquement (les plus récentes d'abord)
