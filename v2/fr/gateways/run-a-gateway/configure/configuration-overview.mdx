## Introduction

Les passerelles dans Livepeer servent de point d'entrée pour<Badge color="blue">Video</Badge> le transcoding et<Badge color="purple">AI</Badge> les workflows de traitement.
Bien que la fonction principale d'ingestion et de distribution des médias soit similaire, les pipelines de traitement et les composants diffèrent.

Une seule passerelle peut gérer les deux types de charges de travail simultanément.

## Méthodes de configuration

Vous avez trois façons de configurer votre passerelle Livepeer après l'installation :

- Options de la ligne de commande (le plus courant)
- Variables d'environnement (précédées de LP_)
- Fichier de configuration (format clé-valeur en texte brut)

## Options de configuration

Cette section décompose l'espace de configuration du passeur Livepeer en les options suivantes :

- **Vidéo & Options de transcodage** - utilisé par les passerelles du réseau vidéo
- **Indicateurs et options de la passerelle IA** - utilisé lors de l'exécution d'une passerelle dans le contexte du réseau IA Livepeer
- **Configuration double** - utilisé lors de l'exécution d'une passerelle qui prend en charge à la fois le transcodage vidéo et les services d'inférence IA
- **Configuration de tarification** - utilisé pour configurer les tarifs des offres de service
