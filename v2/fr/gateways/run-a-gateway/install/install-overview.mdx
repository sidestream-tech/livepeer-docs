## Méthodes d'installation

Installer une passerelle signifie installer le code de la <DoubleIconLink label="go-livepeer" href="https://github.com/livepeer/go-livepeer"/> passerelle.

Vous pouvez soit installer en utilisant

1. <Icon icon="docker" size={20} /> **Docker** (recommandé)
2. Construction à partir de <Icon icon="code" size={20} /> **source (binaire)**
3. Utilisation d'outils développés par la communauté comme <Icon icon="wand-sparkles" size={18} /> **GWID** <Badge color="surface">Beta</Badge> pour une installation et un déploiement en un clic.

## Modes de passerelle

Vous pouvez exécuter une passerelle

- <Icon icon="floppy-disk" size={18} /> **Hors chaîne** -> mode dev ou local
- <Icon icon="link" size={18} /> **Sur chaîne** -> mode de production connecté au réseau Livepeer basé sur la blockchain.{' '}

Si vous prévoyez de fonctionner en mode **En chaîne** mode, veuillez lire les exigences supplémentaires

## Capacités de passerelle

Vous pouvez exécuter une passerelle pour :

- <Badge color="blue"> Video Only </Badge> -> services de transcodage traditionnels
- <Badge color="purple"> AI Only </Badge> -> services d'inférence par IA
- <Badge color="green"> Dual: AI & Video </Badge> -> à la fois le transcodage vidéo et les services d'inférence par IA

<br />
L'installation ci-dessous s'applique aux modes sur chaîne et hors chaîne.{' '}
