## Introduction Les passerelles sont des infrastructures essentielles dans le réseau Livepeer. Elles
fournissent la couche de coordination des services (routage et vérification) qui relie les applications
à la couche de calcul GPU décentralisée (DePIN). Ce guide explique les exigences,
les étapes de configuration et les bonnes pratiques pour exécuter un nœud de passerelle.

## Modes de passerelle

Vous pouvez exécuter une passerelle

- <Icon icon="floppy-disk" size={18} /> **Hors chaîne** -> mode dev ou local
- <Icon icon="link" size={18} /> **Sur la chaîne** -> mode de production connecté au
réseau Livepeer basé sur la blockchain (sur Arbitrum).{' '}

## Capacités de la passerelle

Vous pouvez exécuter une passerelle pour :

- <Badge color="blue"> Video Only </Badge> -> services de transcodage traditionnels
- <Badge color="purple"> AI Only </Badge> -> services d'inférence par IA
- <Badge color="green"> Dual: AI & Video </Badge> -> les services de transcodage vidéo et d'inférence IA
