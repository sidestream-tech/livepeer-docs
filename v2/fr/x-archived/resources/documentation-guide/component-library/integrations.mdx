## Intégration CoinGecko

### Exchanges CoinGecko

Récupère et affiche dynamiquement les échanges qui prennent en charge une pièce à partir de l'API CoinGecko. Affiche un tableau triable des échanges (CEX et DEX) avec des paires de trading, des scores de confiance et des liens.

**Importation :**
```jsx
import { CoinGeckoExchanges } from "/snippets/components/integrations/coingecko.jsx";
```

**Props :**

| Prop | Type | Par défaut | Requis | Description |
|------|------|---------|----------|-------------|
| `coinId` | `string` | `"arbitrum"` | Non | L'ID de pièce CoinGecko (par exemple, "arbitrum", "ethereum", "bitcoin") |

**Fonctionnalités :**
- Récupère automatiquement les données d'échange depuis l'API CoinGecko
- Affiche les échanges dans un tableau triable
- Affiche le type d'échange (CEX/DEX), les paires de trading et les scores de confiance
- Inclut des liens vers les pages de trading
- Prise en charge du tri par nom ou par type
- États de chargement et d'erreur

**Exemples :**

<Tabs>
  <Tab title="Default (Arbitrum)">
    <CoinGeckoExchanges />
  </Tab>
  <Tab title="Ethereum">
    <CoinGeckoExchanges coinId="ethereum" />
  </Tab>
  <Tab title="Code">
```jsx
<CoinGeckoExchanges coinId="arbitrum" />
<CoinGeckoExchanges coinId="ethereum" />
<CoinGeckoExchanges coinId="bitcoin" />
```
  </Tab>
</Tabs>

**Remarque :** Ce composant effectue un appel API en direct à CoinGecko. Assurez-vous que vous ne dépassez pas les limites de taux si vous utilisez plusieurs instances sur une page.
