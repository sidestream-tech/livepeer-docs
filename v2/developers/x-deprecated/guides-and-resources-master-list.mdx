## Part 1 — Guides

How-to guides, tutorials, and walkthroughs for building with Livepeer.

### Getting Started

1. **[Official Quickstart](https://docs.livepeer.org/developers/quick-start)** — Create an API key and send a first stream. Official Livepeer Docs.
2. **[Developer's Guide to Getting Started with Livepeer](https://camiinthisthang.hashnode.dev/the-developers-guide-to-getting-started-with-livepeer)** — Full orientation covering what Livepeer is, how transcoding works, and first API calls. Hashnode, April 2022.
3. **[Liberating Video Infrastructure: A Guide to Decentralisation with Livepeer](https://dev.to/cryptoloom/liberating-video-infrastructure-a-guide-to-seamless-decentralization-with-livepeer-1fgc)** — Architecture walkthrough connecting a developer application to the network. Dev.to, March 2023.

### Video Streaming: Livestreams and VOD

4. **[Create a Livestream (Official Docs)](https://docs.livepeer.org/guides/developing/create-a-livestream)** — Official guide to creating a stream object and playing back via HLS using `useCreateStream` and `useStream`.
5. **[Build a Twitch Clone with the Livepeer API](https://hackernoon.com/build-a-twitch-clone-with-the-livepeer-api)** — Postman-based full stream lifecycle tutorial: create, retrieve, update, record, delete. HackerNoon, March 2022.
6. **[How to Make API Calls with Postman and Livepeer Video Services](https://dev.to/shihyu/how-to-make-api-calls-with-postman-and-livepeer-video-services-15f9)** — Beginner-friendly REST API via Postman, no code required. Dev.to, March 2022.
7. **[How to Stream With Livepeer Video Service's RTMP API](https://dev.to/shihyu)** — Companion tutorial to the Postman guide, using cURL for RTMP streaming. Dev.to, 2022.
8. **[Set Up a Video Player with Video.js and Livepeer](https://github.com/livepeer/livepeer-player-tutorial)** — Official tutorial repo: embed a Livepeer HLS stream into a webpage using Video.js.
9. **[Livepeer Demo App (Next.js)](https://github.com/livepeer/livepeer-demo-app)** — Official demo app demonstrating stream creation and playback using Next.js API routes and the Livepeer.com API.
10. **[Export to IPFS with Livepeer Video Services](https://dev.to/shihyu/export-to-ipfs-with-livepeer-video-services-327d)** — Upload a video asset via the VOD API and export it to IPFS in two API calls. Dev.to, May 2022.
11. **[Upload and Stream Videos off Arweave with Livepeer](https://github.com/livepeer/awesome-livepeer#tutorials)** — Community tutorial: permanent Arweave storage combined with Livepeer transcoding for decentralised video delivery.
12. **[Upload, Transcode, and Stream Videos off IPFS with Livepeer](https://github.com/livepeer/awesome-livepeer#tutorials)** — Fetch an IPFS-hosted video, pass it through Livepeer's transcode pipeline, and serve adaptive bitrate HLS output.

### Video NFTs

13. **[Mint a Video NFT using Livepeer and Polygon](https://dev.to/shihyu/how-to-mint-a-video-nft-using-livepeer-video-services-polygon-1a6g)** — Uses Livepeer's Video NFT SDK to transcode a video and mint it on Polygon's Mumbai testnet. Dev.to, April 2022.
14. **[Livestream API Step-by-Step Demo](https://github.com/camiinthisthang/livepeer-demo-app-instructions)** — Step-by-step walkthrough referenced in the Hashnode getting started guide. GitHub, 2022.
15. **[Video NFT Smart Contract Workshop](https://github.com/camiinthisthang/livepeer-nft)** — Mint a video NFT and create your own smart contract. Referenced in the Hashnode getting started guide. GitHub, 2022.

17. **[Building a Full-Stack Web3 TikTok Clone with React Native and Lens Protocol](https://www.suhailkakar.com/building-a-full-stack-web3-tiktok-clone-with-react-native-livepeer-and-lens-protocol)** — End-to-end mobile app: React Native, `@livepeer/react-native`, Lens Protocol, WalletConnect. March 2023.

Hashnode, March 2023.

Medium, November 2022.
