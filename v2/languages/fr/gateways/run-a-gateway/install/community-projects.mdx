## GWID (Wizard Portail)

<Warning>
  {' '}
  This is a community contributed project. Please contact the team for help & support.{' '}
</Warning>

GWID est une plateforme DevOps entièrement gérée pour Livepeer. Elle fournit un moyen simple et facile pour déployer et gérer des portails Livepeer. Elle fournit également une intégration facile avec d'autres services pour tester le portail.
