Streamplace est une **Entité à Objectif Spécifique (SPE)** dans l'écosystème Livepeer chargée de construire une **couche vidéo décentralisée de bien public, open-source** pour les réseaux sociaux, les plateformes de diffusion en direct et les applications natives Web3.

Elle propose :

- Une **chaîne de traitement de diffusion en direct + pipeline VOD**
- **C2PA + provenance basée sur Ethereum** pour chaque segment vidéo
- Une **schéma de métadonnées** pour les droits, avertissements sur le contenu et les politiques de distribution
- Un **nœud + SDK** pour une intégration facile dans les applications décentralisées
- Un pont vers **Livepeer’s réseau GPU décentralisé**

---

## ✨ Pourquoi Streamplace existe

Les plateformes vidéo centralisées possèdent :

- L'identité
- L'audience
- La distribution
- La monétisation

Streamplace inverse ce modèle avec une infrastructure ouverte, vérifiable et non permissionnée conçue pour :

- Applications clients AT Protocol
- Applications sociales Fediverse
- Créateurs Web3
- Plateformes vidéo gérées par les communautés
