## GWID (Asistente de Gateway)

<Warning>
  {' '}
  This is a community contributed project. Please contact the team for help & support.{' '}
</Warning>

GWID es una plataforma DevOps totalmente gestionada para Livepeer. Proporciona una forma simple y fácil de desplegar y gestionar Livepeer gateways. También ofrece una integración sencilla con otros servicios para probar el gateway.
