## Métodos de instalación

Instalar una Gateway significa instalar el código de la <DoubleIconLink label="go-livepeer" href="https://github.com/livepeer/go-livepeer"/> Gateway.

Puede instalarlo utilizando

1. <Icon icon="docker" size={20} /> **Docker** (recomendado)
2. Construyendo desde <Icon icon="code" size={20} /> **fuente (binario)**
3. Usando herramientas desarrolladas por la comunidad como <Icon icon="wand-sparkles" size={18} /> **GWID** <Badge color="surface">Beta</Badge> para instalación y despliegue de un solo clic.

## Modos de Gateway

Puedes ejecutar un gateway

- <Icon icon="floppy-disk" size={18} /> **Fuera de cadena** -> modo dev o local
- <Icon icon="link" size={18} /> **En cadena** -> modo de producción conectado a la red Livepeer basada en blockchain.{' '}

Si planeas ejecutar en modo**En cadena**modo, por favor lea los requisitos adicionales

## Capacidades de Gateway

Puedes ejecutar un Gateway para:

- <Badge color="blue"> Video Only </Badge> -> servicios tradicionales de transcodificación
- <Badge color="purple"> AI Only </Badge> -> servicios de inferencia de IA
- <Badge color="green"> Dual: AI & Video </Badge> -> tanto servicios de transcodificación de video como servicios de inferencia de IA

<br />
La siguiente instalación se aplica tanto a los modos en cadena como fuera de la cadena.{' '}
