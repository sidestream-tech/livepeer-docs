## Introducción

Los gateways en Livepeer sirven como punto de entrada para <Badge color="blue">Video</Badge> transcodificación y <Badge color="purple">AI</Badge> flujos de trabajo de procesamiento.
Mientras que la función principal de ingestión y distribución de medios es similar, los pipelines y componentes de procesamiento difieren.

Un solo gateway puede manejar ambos tipos de cargas de trabajo simultáneamente.

## Métodos de Configuración

Tienes tres formas de configurar tu gateway Livepeer después de la instalación:

- Bandera de línea de comandos (la más común)
- Variables de entorno (prefijadas con LP_)
- Archivo de configuración (formato de clave-valor en texto plano)

## Opciones de configuración

Esta sección desglosa el espacio de configuración del Gateway Livepeer en las siguientes opciones:

- **Opciones de Video y Transcodificación** - utilizado por los Gateways de Video Network
- **Bandera y Opciones de Gateway AI** - utilizado cuando se ejecuta un Gateway en el contexto de la red AI Livepeer
- **Configuración Dual** - se utiliza al ejecutar un Gateway que admite tanto la transcodificación de video como los servicios de inferencia de IA
- **Configuración de Precios** - se utiliza para configurar los precios de las ofertas de servicio
