# Referencia de la API

¡Bienvenido a la referencia de la API de Livepeer Studio! Aquí encontrarás todos los puntos finales expuestos en la API de Livepeer Studio, aprende cómo usarlos y qué devuelven.

La API de Livepeer Studio está organizada alrededor de REST, tiene URLs orientadas a recursos predecibles, acepta cuerpos de solicitud en JSON, devuelve respuestas codificadas en JSON y utiliza códigos de respuesta HTTP estándar, autenticación y verbos.

## URL base

Todas las solicitudes de API deben realizarse a:

```
https://livepeer.studio/api
```

## Autenticación

Todas las solicitudes requieren autenticación mediante una clave de API. Consulte la [guía de autenticación](../get-started/authentication) para más detalles.

## Formato de respuesta

Todos los resultados son codificados en JSON. Las respuestas exitosas devuelven un `200 OK`código de estado con los datos solicitados en el cuerpo de la respuesta.

Las respuestas de error incluyen un objeto de error con detalles sobre lo que salió mal.
