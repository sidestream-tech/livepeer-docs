## 介绍

在 Livepeer 中，网关作为进入点用于 <Badge color="blue">Video</Badge>转码和 <Badge color="purple">AI</Badge>处理工作流。
虽然摄取和分发媒体的核心功能相似，但处理管道和组件不同。

单个网关可以同时处理这两种工作负载。

## 配置方法

安装后，您有三种方式来配置您的 Livepeer 网关：

- 命令行标志（最常见）
- 环境变量（前缀为 LP_）
- 配置文件（纯文本键值对格式）

## 配置选项

本节将 Livepeer Gateway 配置空间拆分为以下选项：

- **视频与转码选项** - 用于视频网络网关
- **AI Gateway 标志与选项** - 在 Livepeer AI 网络上下文中运行 Gateway 时使用
- **双重配置** - 用于运行支持视频转码和 AI 推理服务的 Gateway
- **定价配置** - 用于配置服务提供的定价
