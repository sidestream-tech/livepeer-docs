## 安装方法

安装网关意味着安装 <DoubleIconLink label="go-livepeer" href="https://github.com/livepeer/go-livepeer"/> 网关代码。

您可以选择使用

1. <Icon icon="docker" size={20} /> **Docker** (推荐)
2. 从<Icon icon="code" size={20} /> **源代码 (二进制)**
3. 使用社区开发的工具如<Icon icon="wand-sparkles" size={18} /> **GWID** <Badge color="surface">Beta</Badge> 用于一键安装和部署。

##  网关模式

您可以运行一个网关

- <Icon icon="floppy-disk" size={18} /> ** 离线** -> 开发或本地模式
- <Icon icon="link" size={18} /> ** 在线** -> 生产模式连接到
基于区块链的 Livepeer 网络。{' '}

如果您计划以**链上** 模式运行，请阅读额外的要求

## 网关功能

您可以运行一个网关用于：

- <Badge color="blue"> Video Only </Badge> -> 传统转码服务
- <Badge color="purple"> AI Only </Badge> -> AI 推理服务
- <Badge color="green"> Dual: AI & Video </Badge> -> 视频转码和 AI
推理服务
