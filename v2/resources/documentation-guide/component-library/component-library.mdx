## How to Use Components

Import components in your MDX files using absolute paths from `/snippets/`:

```jsx
import { YouTubeVideo } from "/snippets/components/content/video.jsx";
import { GotoCard, GotoLink } from "/snippets/components/primitives/links.jsx";
import { CustomCodeBlock } from "/snippets/components/content/code.jsx";
```

Basic UI building blocks used throughout the documentation.

### CustomDivider

A decorative horizontal divider with Livepeer branding icons on both ends. Optionally displays centered text.

<Tabs>

  </Tab>

### Frame Mode Components (PageHeader, H1-H6, P, Divider)

Custom heading and text components for Mintlify frame mode. These components provide styled headings that work properly in frame mode where default markdown headings may not render correctly.

<Tabs>
  <Tab title="Live Example">
    <PageHeader
      title="Page Title"
      subtitle="Page Subtitle"
      description="This is a description of the page content"
    />
    <H1 icon="rocket">Main Heading</H1>
    <H2 icon="star">Section Heading</H2>
    <H3>Subsection</H3>
    <P icon="info-circle">This is a paragraph with an icon.</P>
    <Divider />
  </Tab>
  <Tab title="Code">
```jsx
import { PageHeader, H1, H2, H3, H4, H5, H6, P, Divider } from "/snippets/components/page-structure/frameMode.jsx";

<PageHeader
  title="Page Title"
  subtitle="Page Subtitle"
  description="Page description"
/>

<H1 icon="rocket" iconSize={32}>Main Heading</H1>
<H2 icon="star" align="center">Centered Heading</H2>
<H3>Subsection</H3>
<P icon="info-circle">Paragraph with icon</P>
<Divider margin="2rem 0" />
```
  </Tab>
  <Tab title="Props">
    **PageHeader:**
    | Prop | Type | Default | Description |
    |------|------|---------|-------------|
    | `title` | `string` | required | Main page title |
    | `subtitle` | `string` | - | Subtitle text |
    | `description` | `string` | - | Description text |
    | `titleColor` | `string` | `"var(--hero-text)"` | Title color |
    | `subtitleColor` | `string` | `"var(--accent)"` | Subtitle color |
    | `descriptionColor` | `string` | - | Description color |
    | `children` | `ReactNode` | - | Additional content |

    **H1-H6 (all heading components):**
    | Prop | Type | Default | Description |
    |------|------|---------|-------------|
    | `children` | `string` | required | Heading text |
    | `icon` | `string` | - | Icon name or SVG path |
    | `iconSize` | `number` | varies by level | Icon size in pixels |
    | `iconColor` | `string` | `"var(--accent)"` | Icon color |
    | `align` | `"left" \| "center" \| "right"` | `"left"` | Text alignment |
    | `gap` | `string` | `"0.5rem"` | Gap between icon and text |

    **P (Paragraph):**
    | Prop | Type | Default | Description |
    |------|------|---------|-------------|
    | `children` | `ReactNode` | required | Paragraph content |
    | `icon` | `string` | - | Optional icon |
    | `iconSize` | `number` | `16` | Icon size |
