# Agent Packs

This folder contains generated packs for running the same docs audit pipeline across supported coding agents.

## Pipeline Stages

| Order | Skill ID | Goal |
|---:|---|---|
| 1 | `script-footprint-and-usage-audit` | Detect script sprawl, duplicates, placeholders, and backup artifacts. |
| 2 | `docs-quality-and-freshness-audit` | Surface TODO/TBD/Coming Soon markers, thin content, and freshness risk. |
| 3 | `style-and-language-homogenizer-en-gb` | Enforce UK English and style consistency on English v2 docs in phase 1. |
| 4 | `component-layout-governance` | Validate page composition against component layout contracts by page type. |
| 5 | `cleanup-quarantine-manager` | Classify cleanup candidates and apply conservative quarantine only when requested. |
| 6 | `cross-agent-packager` | Generate consistent skill packs for Codex, Cursor, Claude Code, and Windsurf. |

Generated by: `node tools/scripts/cross-agent-packager.js --agent-pack all`
