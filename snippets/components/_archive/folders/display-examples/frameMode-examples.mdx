<Note>
These components are designed for Mintlify's frame mode pages where default markdown headings may not render correctly. They require importing `ThemeData`.
</Note>

## Headings (H1-H6)

Custom heading components with optional icons and alignment.

### H1 - Main Heading

<H1>Simple H1 Heading</H1>

<H1 icon="rocket">H1 with Icon</H1>

<H1 icon="star" align="center">Centered H1 with Icon</H1>

### H2 - Section Heading

<H2>Simple H2 Heading</H2>

<H2 icon="book">H2 with Icon</H2>

<H2 icon="code" align="center">Centered H2</H2>

### H3 - Subsection Heading

<H3>Simple H3 Heading</H3>

<H3 icon="gear">H3 with Icon</H3>

### H4 - Minor Heading

<H4>Simple H4 Heading</H4>

<H4 icon="lightbulb">H4 with Icon</H4>

### H5 - Small Heading

<H5>Simple H5 Heading</H5>

<H5 icon="info-circle">H5 with Icon</H5>

### H6 - Smallest Heading

<H6>Simple H6 Heading</H6>

<H6 icon="check">H6 with Icon</H6>

## Paragraph (P)

Custom paragraph component with optional icons.

<P>
  This is a simple paragraph without any icon. It uses theme-aware colors.
</P>

<P icon="info-circle">
  This paragraph has an info icon to draw attention.
</P>

<P icon="lightbulb" align="center">
  A centered paragraph with a lightbulb icon for tips.
</P>
