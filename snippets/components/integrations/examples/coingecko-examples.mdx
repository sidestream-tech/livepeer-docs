## CoinGeckoExchanges

Displays a table of exchanges that support a specific cryptocurrency, with sorting, trust scores, and trading links.

### Livepeer Token (LPT)

<CoinGeckoExchanges coinId="livepeer" />

### Ethereum

<CoinGeckoExchanges coinId="ethereum" />

### Arbitrum

<CoinGeckoExchanges coinId="arbitrum" />

## Features

The component includes:

<CardGroup cols={2}>
  <Card title="Sortable Columns" icon="sort">
    Click on "Exchange" or "Type" headers to sort the table
  </Card>
  <Card title="Trust Scores" icon="shield">
    Visual indicators for exchange trust levels (green/yellow/red)
  </Card>
  <Card title="Exchange Types" icon="building">
    Distinguishes between CEX (Centralized) and DEX (Decentralized) exchanges
  </Card>
  <Card title="Direct Links" icon="arrow-up-right">
    Trade links that open directly on each exchange
  </Card>
</CardGroup>

## Usage Notes

<Note>
The `coinId` prop should match the CoinGecko API coin identifier. You can find these IDs on [CoinGecko](https://www.coingecko.com/).
</Note>

### Common Coin IDs

| Token | Coin ID |
|-------|---------|
| Livepeer | `livepeer` |
| Ethereum | `ethereum` |
| Bitcoin | `bitcoin` |
| Arbitrum | `arbitrum` |
| USDC | `usd-coin` |

## Styling

The component uses theme-aware colors:
- Header uses the theme accent color
- Borders adapt to light/dark mode
- Trust score colors are semantic (fixed green/yellow/red)
