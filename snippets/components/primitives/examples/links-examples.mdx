## CustomCallout Component

### Basic Usage

<CustomCallout>
  This is a basic callout with default settings.
</CustomCallout>

### Custom Icon

<CustomCallout icon="rocket">
  This callout uses a rocket icon instead of the default lightbulb.
</CustomCallout>

<CustomCallout icon="warning">
  This is a warning callout with a warning icon.
</CustomCallout>

<CustomCallout icon="info-circle">
  This is an informational callout.
</CustomCallout>

### Custom Color

<CustomCallout color="#3b82f6">
  This callout uses a blue color scheme.
</CustomCallout>

<CustomCallout color="#ef4444">
  This callout uses a red color scheme for warnings.
</CustomCallout>

<CustomCallout color="#8b5cf6">
  This callout uses a purple color scheme.
</CustomCallout>

### Custom Sizes

<CustomCallout iconSize={24} textSize="1rem">
  This callout has larger icon and text.
</CustomCallout>

<CustomCallout iconSize={12} textSize="0.75rem">
  This callout has smaller icon and text.
</CustomCallout>

### All Custom Options

<CustomCallout
  icon="star"
  color="#f59e0b"
  iconSize={20}
  textSize="0.9rem"
>
  This callout combines all custom options: icon, color, and sizes.
</CustomCallout>

## DoubleIconLink Component

### Basic Usage

  label="View on GitHub"
  href="https://github.com/livepeer/go-livepeer"
/>

### Custom Icons

<DoubleIconLink
  label="Read the Docs"

  iconRight="arrow-right"
  href="/docs"
/>

<DoubleIconLink
  label="Watch Tutorial"
  iconLeft="video"

  href="https://youtube.com/example"
/>

<DoubleIconLink
  text="Source code:"
  label="GitHub Repository"
  iconLeft="github"
  href="https://github.com/livepeer/livepeer"
/>

## TipWithArrow Component

### Basic Usage

<TipWithArrow>
  Use the Dropdown at the top-right of this page to view the Quickstart Guide for your preferred OS.
</TipWithArrow>

### Custom Icon

<TipWithArrow icon="rocket">
  This tip uses a rocket icon instead of the default lightbulb.
</TipWithArrow>

<TipWithArrow icon="star">
  Pro tip: Use keyboard shortcuts to navigate faster!
</TipWithArrow>

### Custom Arrow Icon

<TipWithArrow arrowIcon="arrow-right">
  This tip uses a different arrow icon in the corner.
</TipWithArrow>

<TipWithArrow arrowIcon="external-link">
  Click the link icon to open in a new tab.
</TipWithArrow>

### Custom Color

<TipWithArrow color="#3b82f6">
  This tip uses a blue color scheme.
</TipWithArrow>

<TipWithArrow color="#8b5cf6">
  This tip uses a purple color scheme.
</TipWithArrow>

### Custom Sizes

<TipWithArrow iconSize={20} arrowSize={20}>
  This tip has larger icons (20px instead of default 16px).
</TipWithArrow>

<TipWithArrow iconSize={12} arrowSize={12}>
  This tip has smaller icons for subtle hints.
</TipWithArrow>

### All Custom Options

<TipWithArrow
  icon="star"
  arrowIcon="arrow-up-right-from-square"
  color="#f59e0b"
  iconSize={18}
  arrowSize={14}
>
  This tip combines all custom options: custom icon, arrow, color, and sizes.
</TipWithArrow>
