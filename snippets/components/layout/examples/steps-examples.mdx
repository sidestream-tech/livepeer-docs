## Installation Guide

<StyledSteps>
  <StyledStep title="Prerequisites" icon="list-check">
    Ensure you have the following installed:
    - Node.js 16 or higher
    - npm or yarn package manager
    - A code editor (VS Code recommended)
  </StyledStep>
  <StyledStep title="Install Package" icon="download">
    ```bash
    npm install @livepeer/react
    ```
    Or using yarn:
    ```bash
    yarn add @livepeer/react
    ```
  </StyledStep>
  <StyledStep title="Import Components" icon="file-import">
    ```javascript
    import { LivepeerConfig, Player } from '@livepeer/react';
    ```
  </StyledStep>
  <StyledStep title="Configure Provider" icon="gear">
    ```javascript
    <LivepeerConfig client={client}>
      <YourApp />
    </LivepeerConfig>
    ```
  </StyledStep>
  <StyledStep title="Start Using" icon="play">
    You're all set! Start building with Livepeer components.
  </StyledStep>
</StyledSteps>

## API Integration Steps

<StyledSteps
  iconColor="#dc2626"
  titleColor="#ef4444"
  lineColor="#f87171"
>
  <StyledStep title="Authentication" icon="shield-halved">
    Include your API key in the request headers:
    ```
    Authorization: Bearer YOUR_API_KEY
    ```
  </StyledStep>
  <StyledStep title="Make Request" icon="paper-plane">
    Send a POST request to create a stream:
    ```
    POST https://livepeer.studio/api/stream
    ```
  </StyledStep>
  <StyledStep title="Handle Response" icon="inbox">
    Parse the JSON response to get your stream details
  </StyledStep>
  <StyledStep title="Start Streaming" icon="video">
    Use the streamKey to start broadcasting
  </StyledStep>
</StyledSteps>

## Quick Start

<StyledSteps>
  <StyledStep title="Clone Repository" icon="code-branch">
    ```bash
    git clone https://github.com/livepeer/example-app.git
    cd example-app
    ```
  </StyledStep>
  <StyledStep title="Install Dependencies" icon="box">
    ```bash
    npm install
    ```
  </StyledStep>
  <StyledStep title="Configure Environment" icon="file-code">
    ```bash
    cp .env.example .env
    # Edit .env with your API key
    ```
  </StyledStep>
  <StyledStep title="Run Development Server" icon="play">
    ```bash
    npm run dev
    ```
  </StyledStep>
</StyledSteps>
