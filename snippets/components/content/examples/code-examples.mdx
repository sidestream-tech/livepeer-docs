## CustomCodeBlock

### Basic Usage

<CustomCodeBlock
  filename="example.js"
  icon="javascript"
  language="javascript"
  codeString={`
function hello() {
  console.log("Hello, World!");
}
`}
/>

### With Placeholder Replacement

<CustomCodeBlock
  filename="config.yaml"
  icon="file"
  language="yaml"
  codeString={`
api_key: {PLACEHOLDER}
endpoint: https://api.example.com
`}
  placeholderValue="your-api-key-here"
/>

### With Pre and Post Notes

<CustomCodeBlock
  filename="install.sh"
  icon="terminal"
  language="bash"
  preNote="Run this command to install dependencies:"
  postNote="This may take a few minutes to complete."
  codeString={`npm install @livepeer/react`}
/>

### With Expected Output

<CustomCodeBlock
  filename="test.js"
  icon="javascript"
  language="javascript"
  codeString={`console.log("Testing output");`}
  output={{
    codeString: "Testing output",
    filename: "Console Output",
    icon: "terminal",
    language: "bash"
  }}
/>

## ComplexCodeBlock

### With Pre Note

<ComplexCodeBlock
  filename="deploy.sh"
  icon="rocket"
  language="bash"
  preNote="⚠️ Make sure you have proper permissions before running this script."
  codeString={`
#!/bin/bash
npm run build
npm run deploy
`}
/>

### With Post Note

<ComplexCodeBlock
  filename="result.json"
  icon="file-code"
  language="json"
  postNote="The response will be cached for 5 minutes."
  codeString={`
{
  "status": "success",
  "data": {}
}
`}
/>

### With Both Notes and Highlighting

<ComplexCodeBlock
  filename="important.js"
  icon="exclamation-triangle"
  language="javascript"
  highlight="2,3"
  preNote="Pay attention to the highlighted lines:"
  postNote="These lines are critical for proper functionality."
  codeString={`
function process() {
  const result = calculate();
  return validate(result);
}
`}
/>

## CodeSection

### Using CodeSection Wrapper
