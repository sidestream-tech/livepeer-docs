## ShowcaseCards

A filterable, paginated gallery of project cards with search, category filters, and product filters.

### Basic Usage

<ShowcaseCards
  items={[
    {
      title: "StreamHub",
      subtitle: "Live Streaming Platform",
      description: "A decentralized streaming platform for content creators.",
      href: "https://example.com/streamhub",
      categoryTags: ["Video Streaming"],
      productTags: ["Livepeer Studio"],
      links: [
        { website: "https://streamhub.example.com" },
        { github: "https://github.com/example/streamhub" },
      ],
    },
    {
      title: "VideoAI",
      subtitle: "AI Video Processing",
      description: "Apply AI models to video content in real-time.",
      href: "https://example.com/videoai",
      categoryTags: ["AI-Powered Apps"],
      productTags: ["Livepeer Network"],
      links: [
        { website: "https://videoai.example.com" },
      ],
    },
    {
      title: "SocialStream",
      subtitle: "Social Video Network",
      description: "Share and discover video content with your community.",
      href: "https://example.com/socialstream",
      categoryTags: ["Social Media"],
      productTags: ["Livepeer Studio"],
      links: [
        { x: "https://x.com/socialstream" },
        { discord: "https://discord.gg/socialstream" },
      ],
    },
    {
      title: "DevTools",
      subtitle: "Developer Utilities",
      description: "Tools for building on the Livepeer network.",
      href: "https://example.com/devtools",
      categoryTags: ["Developer Tools"],
      productTags: ["Livepeer Network"],
      links: [
        { github: "https://github.com/example/devtools" },
      ],
    },
  ]}
/>

## With Pagination

When you have many items, pagination is automatically enabled:

<ShowcaseCards
  pageSize={2}
  items={[
    {
      title: "Project 1",
      subtitle: "Description 1",
      description: "First project in the showcase.",
      categoryTags: ["Apps"],
      productTags: ["Livepeer Studio"],
    },
    {
      title: "Project 2",
      subtitle: "Description 2",
      description: "Second project in the showcase.",
      categoryTags: ["Apps"],
      productTags: ["Livepeer Network"],
    },
    {
      title: "Project 3",
      subtitle: "Description 3",
      description: "Third project in the showcase.",
      categoryTags: ["Community"],
      productTags: ["Livepeer Studio"],
    },
    {
      title: "Project 4",
      subtitle: "Description 4",
      description: "Fourth project in the showcase.",
      categoryTags: ["Community"],
      productTags: ["Livepeer Network"],
    },
  ]}
/>

## Features

<CardGroup cols={2}>
  <Card title="Search" icon="magnifying-glass">
    Filter projects by title, subtitle, or description
  </Card>
  <Card title="Category Filter" icon="tags">
    Filter by category (Apps, Social Media, etc.)
  </Card>
  <Card title="Product Filter" icon="filter">
    Filter by Livepeer product integration
  </Card>
  <Card title="Pagination" icon="chevron-right">
    Navigate through large collections
  </Card>
</CardGroup>

## Category Tags

Available category colors:

| Category | Badge Color |
|----------|-------------|
| Apps | Blue |
| Social Media | Purple |
| Video Streaming | Green |
| Community | Yellow |
| Marketplaces | Orange |
| Developer Tools | Red |
| AI-Powered Apps | Cyan |

---

## Product Tags

Projects can be tagged with Livepeer products:

- Livepeer Studio
- Livepeer Network
- Daydream
- Stream.place
- Frameworks

Each product tag includes a link to its documentation.
