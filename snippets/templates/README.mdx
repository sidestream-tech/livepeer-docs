Use this folder for copy-ready authoring scaffolds that contributors can duplicate into `v2/` pages or reuse inside existing MDX files.

## Page Templates

- `pages/faq-page.mdx`
- `pages/troubleshooting-page.mdx`
- `pages/openapi-endpoint-page.mdx`

## Block Templates

- `blocks/related-pages-cta.mdx`
- `blocks/comparison-matrix.mdx`

## Usage

1. Copy the closest template into the target docs path.
2. Replace the placeholder title, description, and body content.
3. Keep the canonical OG metadata and update `status` and `lastVerified` when the page becomes current.
