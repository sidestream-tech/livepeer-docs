# Livepeer Snippets Library

This folder contains all custom components used in the Livepeer documentation.

## Overview
Mintlify uses the Snippets folder to store all custom components used in the documentation.
Mintlify allows React Components to be used in MDX files - with some caveats.

**Mintlify Gotchas**
- You cannot import components or data into another component
    You must have the import in the MDX file that uses the component
