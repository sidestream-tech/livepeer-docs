# AI Jobs & Pipelines Quickstart

Start here if you want to send AI inference jobs through a Livepeer gateway.

## Choose a quickstart

- [AI Jobs quickstart](./ai-jobs) - End-to-end request flow using the AI gateway API (GitHub-validated endpoint surface; stakeholder flow signoff still required).
- [AI Pipelines overview](/developers/ai-pipelines/overview) - Broader overview of the AI pipeline ecosystem on Livepeer.
- [BYOC (Bring Your Own Compute)](/developers/ai-pipelines/byoc) - Run your own compute and connect it to the Livepeer AI network.
