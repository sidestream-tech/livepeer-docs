Livepeer Studio is a developer-friendly gateway and hosted platform for the Livepeer network. Use it to add **live streaming** and **video-on-demand (VOD)** to your application with minimal setup.

<Info>
 For Livepeer's hosted real-time StreamDiffusion AI platform **Daydream**, see [Daydream](https://pipelines.livepeer.org/docs) and the [Daydream product page](/solutions/daydream/overview).
</Info>

## What you can do

- **Livestream** - Create streams, ingest via RTMP or WebRTC, play back with low-latency WebRTC or HLS, record sessions, and multistream to other platforms.
- **Video on demand** - Upload assets (TUS or PUT), transcode, play back with the Livepeer Player or any HLS/MP4 player, and optionally use encrypted assets.
- **Access control** - Gate playback with webhooks or JWTs (e.g. subscriptions, token-gating).
- **Events and analytics** - Use webhooks for stream/asset events and the viewership API for engagement and performance metrics.

## SDKs and API

- **Server SDKs** - [TypeScript/JavaScript](https://www.npmjs.com/package/livepeer), [Go](https://pkg.go.dev/github.com/livepeer/livepeer-go), [Python](https://pypi.org/project/livepeer/) for calling the Livepeer Studio API.
- **React** - [@livepeer/react](https://www.npmjs.com/package/@livepeer/react): Player and Broadcast components for playback and in-browser streaming.
- **API** - REST API for streams, assets, playback, webhooks, multistream, and more. See [API overview](/solutions/livepeer-studio/reference/api) and the [Livepeer Studio API docs](https://livepeer.studio/docs).
